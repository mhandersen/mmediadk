Welcome...

M media ApS have existed since 1999 delivering web solutions based on the latest technology. Focus is on the following areas:

Hosting websites and providing Virtuel Server technologies to our customers. We have hosted ASP.NET websites since 2003 and have extensive knowledge of the Microsoft .NET platform.

Web development using the Microsoft ASP.NET platform and offer a wide range of web related services.

What can we do for you:

Services

.NET hosting

We specialize in ASP.NET and .NET Core hosting and host our sites using Microsoft Windows servers and IIS.

We have experience hosting a number of .NET based systems like: Umbraco, nopCommerce, DNN, Composite C1, Orchard Project, mojoPortal etc.

Visit www.aspnethotel.dk for more information.

Virtual Servers

We have been providing Microsoft Windows-based Virtual Servers to our customers since 2005.

Our customers typical use theirs servers to host large webshops, run customized systems build using the .NET framework, provide CMS solutions with thousands of pages and visitors etc.

Visit www.virtualprivateserver.dk for more information.

Web development

We do Web development using the Microsoft ASP.NET platform and offer a wide range of web related services:

Integrated web solutions/sites, Ecommerce with creditcard payment- and integration, programming and scripting, Content Management Systems (CMS) etc.

Wordpress hosting

Our customers asked us to host their WordPress sites for years. In 2015 we finally decided it was time to host the #1 CMS in the world.

We use Linux servers to host our WordPress sites and are able to help our customers with questions like which plug-in to use, how to move an entire site, content creation etc.
