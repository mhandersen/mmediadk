Velkommen...

M media ApS har siden 1999 leveret webløsninger baseret på den nyeste teknologi. Fokus er på følgende områder:

Hosting af hjemmesider og Virtuelle Servere til vores kunder. Vi har hosted ASP.NET websites siden 2003 og har stor viden omkring Microsoft .NET platformen.

Udviklingydelser med speciale indenfor Microsoft .NET platformen og relaterede services.

Hvad vi kan hjælpe dig med:

Ydelser

.NET hosting

Vi har specialiseret os i ASP.NET og .NET Core hosting og anvender udelukkende Microsoft Windows servers og IIS til dette formål.

Vi har stor erfaring med hosting af en række .NET baseret systemer som: Umbraco, nopCommerce, DNN, Composite C1, Orchard Project, mojoPortal m.fl.

Besøg www.aspnethotel.dk for mere information.

Virtuelle Servere

Vi har leveret Microsoft Windows-baseret Virtuelle Servere til vores kunder siden 2005.

Vores kunder anvender typisk deres servere til at drifte store webshops, special udviklet systemer på .NET platformen, CMS løsninger med tusindvis af sider og mange aktive besøgende m.m.

Besøg www.virtualprivateserver.dk for mere information.

Webudvikling

Vi tilbyder udviklingydelser med speciale indenfor Microsoft ASP.NET platformen, herunder blandt andet:

Integrerede webløsninger, administrations- og betalingssystemer, kundestyringssystemer (CRM), programmering og scripting, Content Management Systemer (CMS) m.m.

Wordpress hosting

Vores kunder efterspurgte WordPress hosting i flere år - og i 2015 begyndte vi at drifte dette CMS for nye og eksisterende kunder.

Vi anvender Linux servere til at hoste WordPress og kan hjælpe vores kunder med spørgsmål vedr. plug-in's, hvordan man flytter sit WP site, udarbejdelse af indhold m.m.
